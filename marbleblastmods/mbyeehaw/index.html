Marble Blast Yeehaw

Marble Blast Yeehaw is a Marble Blast modification created by pasta_real. It was planned to be the next Marble Blast mod after Marble Blast Wonder with new features and fixing issues (obnoxious noises, low effort,..) before it was merged into Marble Blast Wildron. Only 1 build currently exists, and it is nearly identical to Marble Blast Gold.
