Marble Blast Next

Marble Blast Next was a modification made back in 2017 by NeoTheLynx. It was supposed to be the "next-generation" of Marble Blast, hence the name Marble Blast Next. It used the new Torque3D engine which is an upgraded version of Torque Game Engine. The mod poses similarity to Marble Blast Ultra, mostly because NeoTheLynx was in the Marble Blast Ultra PC Port team. Be warned when playing this mod as it is quite unoptimized.
